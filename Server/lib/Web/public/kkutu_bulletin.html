5.8.0 [210131]

  • 1차 클랜 기능 개편 작업을 진행했습니다.

  • 아래와 같은 내용들이 변경, 개선되었습니다.

  • 1. 여러 피드백을 받아들여, 클랜원 목록 부분을 개선했습니다.
  • 기존 식별 번호만 표시하던 방식에서 식별 번호, 닉네임, 직위 모두 표시하도록 개선했습니다.
  • 향후 실시할 2차 클랜 기능 개편 작업에서는 클랜원 추방을 더 쉽게 할 수 있도록 개선할 것이고
  • 클랜원을 클랜 부마스터로 승급, 클랜 부마스터를 클랜원으로 강등할 수 있는 기능을 추가할 예정입니다.

  • 2. 보안을 위해 클랜 생성 시 클랜 관리에 사용할 비밀번호를 입력해야 합니다.
  • 입력한 비밀번호는 단방향 암호화 방식을 통해 암호화되어 저장되므로
  • 타인은 물론 DB 권한을 가진 관리자도 저장된 비밀번호가 무엇인지 알 수 없습니다.
  • 하지만, 비밀번호를 잊는다면 관리자에게 문의해도 소용없으니 주의해야 합니다.
