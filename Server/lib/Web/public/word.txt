BF끄투 단어 등재기준

최대 등재 가능한 글자수: 한글 60, 영어 70

기호 및 문자 읽는 방법 : %(퍼센트), &(앤드), vs(대)

한국어 단어 등재

- 영어가 들어갔을 경우는 공식 표기를 원칙으로 하며, 없을 경우에는 표준 영어발음으로 등재한다.

-다음의 단어 등급으로 방어단어 갯수별로 단어의 추가 여부가 결정된다.-

1. 추가금지단어 : 공격단어가 방어단어보다 많고, 방어단어가 11개 이하인 경우(우리말샘 단어 포함)
이 경우에는 방어단어만 추가 가능하다. (단, 공격/방어 단어가 서로 11개 이하인 경우에는 비율을 맞추어가면서 추가할 수 있다.)
2. 방단제시단어 : 공격단어가 방어단어보다 많고, 방어단어가 12~79개 이하인 경우(우리말샘 포함)
이 경우에는 공격단어를 추가할 수 있다. 단, 그에 상응하는 방어단어를 제시하여야 한다.(비율별) 
공단/방단의 비율이 1:1~2:1의 경우 : 방어단어 1개
공단/방단의 비율이 3:1~4:1의 경우 : 방어단어 2개
공단/방단의 비율이 5:1~9:1의 경우 : 방어단어 3개
공단/방단의 비율이 10:1 이상의 경우 : 방어단어 4개
단, 추가 방어단어가 없다고 판단될 시 일부 동사나 형용사 추가를 허용토록 한다. 그러나, 만일 스택 방어 단어가 앞뒤가 똑같은 단어 (예 : 퀑퀑)만 존재하고 다른 방어단어가 없을 시에는 1방화 시킨다
3. 자율형 추가단어 : 공격단어, 방어단어 모두 80개 이상인 경우(우리말샘 단어 포함)
이 경우에는 비율에 상관없이 어떤 단어든 추가할 수 있다.
4. 1방단어 : 1방단어는 앞/끝, 공단/방단 비율 상관없이 어떤 단어든 추가할 수 있다.

예외사항
1. 현재 스택 단어에서 더 이상의 방어 단어가 없을 경우, 일부의 동사나 형용사를 허용한다.
2. 현재 스택 단어 중 악용 가능성이 높은 스택 단어이면서 방어 단어가 더이상 없고 동사나 형용사도 없을 시 한방 단어로 처리한다.
3. 기호 &(앤드)표시가 공식적으로 '앤'표시일 경우 '앤'으로 등재한다.
4. 어인정으로만 쓸 수 있는 공/방 단어일 경우 가장 많은 쪽을 우선으로 등재한다. (예시 : 뭐, 뭘, 봐)

영어 단어 등재

- 영어 단어 등재는 소문자로만 하며 띄어쓰기, 기호는 무시한다. (즉, 띄어쓰기 없이 붙여서 등재)
- & 기호의 표기는 and로 한다.
- 네이버 영어사전, 워드넷, 윅셔너리에 포함된 뜻이 있는 단어만 등재한다.
- 뜻에 약자, 웹수집 (네이버 지식백과는 가능), 번역기 뜻(정식 뜻이 아닌 것)은 제외
